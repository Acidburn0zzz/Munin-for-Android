NAME

nut_misc - Plugin to monitor UPS via the upsc command

CONFIGURATION

The following configuration parameters are used by this plugin

 [nut_misc]
  env.upsname - <name@host>
  env.upsc    - <command>

DEFAULT CONFIGURATION

 [nut_misc]
  env.upsname bertha@127.0.0.1
  env.upsc    upsc
