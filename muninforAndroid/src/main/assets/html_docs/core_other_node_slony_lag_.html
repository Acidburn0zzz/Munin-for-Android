NAME

slony_lag_ - Plugin to monitor slony replication lag.

CONFIGURATION

Configuration is done through libpq environment variables, for example PGUSER, PGDATABASE, etc. For more information, see Munin::Plugin::Pgsql.

To monitor a specific database, link to slony_lag_<databasename>:<clustername>.
