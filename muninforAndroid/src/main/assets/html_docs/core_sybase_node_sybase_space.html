NAME

sybase_space - Plugin to monitor sybase database space usage

CONFIGURATION

You need to add the user to all the databases you want monitored.

Configuration variables:

 SYBASE      - Sybase home
 SYBASE_USER - Username
 SYBASE_PASS - Password
 SYBASE_HOST - Host
