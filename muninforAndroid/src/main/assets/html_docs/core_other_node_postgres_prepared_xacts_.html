NAME

postgres_prepared_xacts_ - Plugin to monitor number of prepared transactions in PostgreSQL.

CONFIGURATION

Configuration is done through libpq environment variables, for example PGUSER, PGDATABASE, etc. For more information, see Munin::Plugin::Pgsql.

To monitor a specific database, link to postgres_prepared_xacts_<databasename>. To monitor all databases, link to postgres_prepared_xacts_ALL. To monitor several instances, link to postgres_<tag>_prepared_xacts_<databasename|ALL> The <tag> can be what you want but without "_". It allows you to define several database configuration.

Example : [postgres_pg91_*] env.PGPORT 5432 [postgres_pg92_*] env.PGPORT 5432
