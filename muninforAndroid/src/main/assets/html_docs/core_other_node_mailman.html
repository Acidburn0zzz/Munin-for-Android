NAME

mailman - Plugin to monitor activity on mailman mailing list management systems

CONFIGURATION

Configuration for this plugin depends on the mailman's settings on the system. Here are the defaults:

[mailman] env.logfile /var/log/mailman/post env.libdir /var/lib/mailman env.lister /usr/lib/mailman/bin/list_members
